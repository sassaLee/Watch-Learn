# Watch-Learn: Schema do Banco de Dados PostgreSQL

Este repositório contém o código SQL completo da arquitetura do banco de dados para o aplicativo Watch&Learn. 

O projeto demonstra:
* Criação das 4 Tabelas Relacionais (DDL)
* Lógica de Automação de Perfil (Função e Trigger)
* Regras de Segurança por Linha (RLS - Policies)
